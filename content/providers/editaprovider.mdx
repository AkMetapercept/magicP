# Edit a Provider

You can edit the Provider details of a particular Provider that is already created.

To edit the Provider in MagicPixel, perform the following steps:
1.	Select the Provider that you want to edit.
2.	From the **Actions** drop-down menu on the top right corner, select **Edit**.

![editaprovider](/images/providersimages/edit.png)

3. Edit the Provider details and click **Save** on the top-right to save the changes.
