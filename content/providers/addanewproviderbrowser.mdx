# Add a New Provider (For Browser/Client-side)
For Client-side or Browser-side configuration, all the information is fetched from the webpage or browser.

To add a new Provider on MagicPixel, perform the following steps:
1. From the left navigation panel, select **Providers**.

![addanewprovider](/images/providersimages/providers.png)

2. Select **Add Provider -> For Browser** on the top-right corner of the screen.

![addanewprovider](/images/providersimages/addproviderbrowser.png)

3. Select a **Provider Template**.

![addanewprovider](/images/providersimages/providertemplate.png)

4.	Enter the Provider Details such as Provider Name and Description in the New Provider screen.
5.	The Script section includes both the Template script and Provider script. 

![addanewprovider](/images/providersimages/script.png)

You can enter the JavaScript in the Provider Script section and click **Save** to save the changes.
6.	The **Attributes** section lists the Attributes for Data Elements and Transformers. This is an Optional section.

![addanewprovider](/images/providersimages/attributes.png)

7.	Select **Choose Data Element** for Data Element attributes.
8.	Select either User-Defined Data Element or Default Data Element.

![addanewprovider](/images/providersimages/selectdataelement.png)

**Step Result:** The Data Element selected is visible under the Value section.

![addanewprovider](/images/providersimages/attributesoptional.png)

9.	Select **Choose Transformer** for Data Element Attributes.

![addanewprovider](/images/providersimages/transformer.png)

**Step Result**: The Transformer selected is visible under the Value section.
