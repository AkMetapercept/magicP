# Edit a Transformer

You can edit the Qualification Criteria details of a particular criteria that is already created.

To edit the Transformer in MagicPixel, perform the following steps:

1.	Select the Transformer that you want to edit.
2.	From the **Actions** drop-down menu on the top right corner, select **Edit**.

![editatransformer](/images/transformersimages/edit.png)

3.	You can edit the details under Info section, Function Arguments section, and Code section and click **Save** to save the changes.
