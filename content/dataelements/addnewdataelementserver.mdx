# Add a New Data Element (For Server-side)

For Server-side configuration, all the information is fetched from the server.

Perform the following steps to create the Data elements on browser:
1.	Click **Add Data Element (For Server)** on the top right corner of the screen.

![addnewdataelementsserver](/images/dataelementsimages/adddataelementserver.png)

2.	Enter the following details in the New Data Element screen that appears.
3.	Select the Check box Default Data Elements to list the default data elements that are readily available in MagicPixel. The specific default template that is available will be used for the particular data elements.

![addnewdataelementsserver](/images/dataelementsimages/dataelements.png)

4.	Update the Info section on the top-left corner of the screen.

![addnewdataelementsserver](/images/dataelementsimages/newdataelementserver.png)

5.	Enter the following fields:

| Field Name      | Description |
| ----------- | ----------- |
| Name      |  The Default Data Element Name.  |
| Variable Name  | Denotes the variable name. <br /> **Note**: The Variable name is same as the Data Element name. |
| Description |  Enter the description of the Data Element. |
| Choose Data Element Type | Denotes the data element type: <br /> • **Server Cookie**: <br /> • **Server Header**: <br /> • **JSON path**:  |

The following screen appears if you select Server Cookie:

![addnewdataelementsserver](/images/dataelementsimages/servercookie.png)

6. Enter the Cookie name.

7.	Select the Default Value from the drop-down. If the value is not available for a particular tag, then default value is used.
- **Undefined**: If the data element is not supposed to be used then select Undefined.
- **Empty String**: Denotes that the data element is a string value.
- **Null**: Denotes the Null data type for the data element.
- **Custom Value**: Denotes the static value for the Data Element.

8.	Click **Save** to save the changes.
